# Superstore Sales Analysis Dashboard





#### Project Overview



This project analyzes retail sales data to identify business insights related to sales performance, profitability, product categories, and regional trends.



#### The project was completed using:



* &#x20;Excel for data cleaning
* &#x20;Python for data processing and analysis
* Power BI for dashboard creation and visualization





#### Tools \& Technologies Used



* Microsoft Excel
* Python
* Pandas
* Power BI
* DAX





### Project Workflow



#### 1\. Data Cleaning using Excel



* &#x20;Removed duplicates
* &#x20;Handled missing values
* &#x20;Corrected formatting issues
* Prepared clean dataset for analysis



#### 2\.Data Processing using Python



* &#x20;Loaded dataset using Pandas
* &#x20;Created month-wise analysis
* &#x20;Performed data transformation
* Generated analytical tables



### 3\. Dashboard Creation using Power BI



#### Created an interactive dashboard with:



* &#x20;KPI Cards
* &#x20;Sales Trend Analysis
* &#x20;Profit Analysis
* &#x20;Regional Performance Analysis
* &#x20;Category-wise Sales Distribution
* &#x20;Interactive Filters





### &#x20;Dashboard Features



#### KPI Metrics



* &#x20;Total Sales
* &#x20;Total Profit
* &#x20;Total Orders
* &#x20;Profit Margin



#### Visualizations



* &#x20;Sales by Region
* &#x20;Profit by Category
* &#x20;Monthly Sales Trend
* &#x20;Sales by Sub-Category
* &#x20;Sales vs Profit Scatter Plot
* &#x20;Category-wise Sales Distribution





#### Interactive Filters



* &#x20;Region Filter
* &#x20;Category Filter
* &#x20;Date Filter





#### &#x20;Key Insights



* &#x20;West region generated the highest sales
* &#x20;Technology category was the most profitable
* &#x20;Sales peaked during November and December
* &#x20;Some sub-categories showed high sales but low profit
